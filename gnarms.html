[HG] GN Arms Type E Real Color ver.

Harga Rp 1.200.000,00

Mint in Box Bandai HG [High Grade] version dgn skala 1/144. Set lengkap dari box, buku manual dan plastic modelnya.

Deskripsi:

The GNR-001 GN Arms Type E (aka GN Arms E, GN Arms) adalah sebuah mobile armor/weapons yg diciptakan untuk Gundam Exia. GN Arms nya sendiri dipiloti oleh Lasse Aeon. Didesain dari data Gundam Sefer Rasiel dan Avalanche Exia. GN Arms sendiri merupakan unit yg diproduksi terbatas, hanya ada dua macam GN Arms (Type D dan Type E), dimana keduanya dimodifikasi sesuai dgn karakteristik pertempuran dari Gundam yg menggunakannya. Persenjataan-nya terdiri dari sepasang Large Size GN Cannon dan GN Claws. Khusus utk GN Arms Type E, dilengkapi dgn sepasang Large Size GN Sword dan GN Beam Guns.

Untuk set model kit nya sendiri, set ini hanya berisi GN Arms Type E (tidak termasuk Gundam Exia). Lalu apa perbedaannya dgn GN Arms Type E yg regular? GN Arms Type E Real Color Ver. mendapatkan sentuhan detailing khususnya dari decal yg ada dalam set ini. Selain itu, GN Arms yg satu ini juga fit untuk RG Gundam Exia. Sangat cocok untuk dikoleksi bersamaan dengan RG Gundam Exia!
